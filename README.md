# PIC-Microcontroller-Programming-Peripheral-Interfaces
This repository contains basic programs and experiments performed while learning PIC microcontroller programming.

All programs were developed and tested using MPLAB IDE to understand microcontroller architecture, peripheral control, and communication protocols.

🔧 Topics Covered

Timer configuration

Counter implementation

Interrupt-based programming

Basic microcontroller projects

Serial communication concepts

Peripheral communication using I2C, UART, USART and SPI

🛠 Tools Used

MPLAB IDE

PIC Microcontroller

Embedded C

🎯 Purpose

This repository documents my learning journey in microcontroller programming and embedded communication systems, forming the base for more advanced embedded and IoT projects.
